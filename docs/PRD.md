# Portfolio Website for Pham Huynh Quoc Thang

## Project Overview

Tạo trang portfolio ấn tượng cho Front-End Developer sử dụng React + Vite + TypeScript + TailwindCSS + Framer Motion.

## Problem Statement

Cần một trang portfolio cá nhân để showcase kinh nghiệm làm việc, projects, và technical skills cho nhà tuyển dụng.

## Goals

1. Tạo trang portfolio với thiết kế hiện đại, dark theme
2. Showcase 3 vị trí công việc: Arobid, Amazing Tech, FPT Software
3. Hiển thị 2 personal projects: Shopee Clone, Twitter Clone
4. Responsive design cho tất cả devices
5. Animations mượt mà với Framer Motion

## User Stories Summary

### Epic 1: Project Foundation
- Story 1.1: Initialize React + Vite + TS project
- Story 1.2: Configure TailwindCSS + Design System
- Story 1.3: Setup Framer Motion + Base Animations

### Epic 2: Core Components
- Story 2.1: Hero Section with Animated Text
- Story 2.2: Navbar with Smooth Scroll
- Story 2.3: About Section

### Epic 3: Experience & Projects
- Story 3.1: Experience Timeline Component
- Story 3.2: Projects Showcase Cards
- Story 3.3: Skills Grid with Icons

### Epic 4: Contact & Polish
- Story 4.1: Contact Section
- Story 4.2: Responsive Design & Mobile
- Story 4.3: Performance Optimization
